**X-window**

* 리눅스용 X-window는 XFree86 프로젝트에서 만들었다.
* 1984년 개발
* 아데나 프로젝트
* X 컨소시엄에 의해 X11버전이 처음 개정되어 X11R2가 발표
* X 윈도우는 디스플레이 장치에 의존적이지 않고 서로 다른 기종을 함께 사용할 수 있다.
* X11R6.3을 끝으로 해체
* 기본 6000번 포트 사용
* 서로간의 통신을 위해 X Protocol 사용

* 
* X server : 그래픽하게 보여주는 하나의 시스템
* X client : x server 위에 동작하는 응용프로그램
  * xhost : X server에 접속할 수 있는 클라이언트를 지정하거나 해제 , IP로 접속
  * xauth : 키 값을 이용해서 접속
    * xauth list $DISPLAY

**리눅스 파일공유**

* NFS (Network File System)
  * 네트워크 상에서 **다른 컴퓨터**의 파일시스템을 **마운트하여** 사용한다.
  * RPC :  NFS의 프로토콜, 네트워크 파일 시스템
* NIS(Network Information Service)
  * 네트워크 정보 서비스를 가동한다.
* CIFS 
  * 윈도우 공유폴더에 사용되는 프로토콜
  * 리눅스와 윈도우 서버를 혼용해서 사용하는 경우 사용
  * TCP/IP 프로토콜 기반
  * SAMBA 프로그램과 관련
* SAMBA
  * 윈도우를 사용하는 pc에서 리눅스 또는 유닉스 서버에 접속하여 사용하게 도와주는 프로그램
* telnet
  * 인터넷을 통한 원격 접속
  * 인터넷이나 로컬 영역 네트워크 연결에 쓰이는 네트워크 프로토콜
  * 원격으로 리눅스 시스템에 접속하여 사용할 때

**메일 서버 프로토콜**

**통신**

* TCP : 전화 같은 역할, 받는 사람을 정하고 시작
  * 전송 제어 프로토콜
  * 응답을 주고 받는다.

* UDP : 편지 같은 역할, 
  * 보내기만 하고 응답을 주고 받지 않는 프로토콜, 전달 확인도 안함. 
* ICMP 
  * 인터넷 제어 메세지 프로토콜
  * 오류 메세지를 전송하는데 주로 쓰임
* ARP
  * 주소 결정 프로토콜
  * IP주소를 물리적 네트워크 주소로 대응
